## Qt_MusicPlayer

A simple tutorial project: a minimal music player built with C++/Qt and QML.

Qt_MusicPlayer is an educational/demo project demonstrating how to build a basic audio player using the Qt 6 framework.
The project combines backend logic written in C++ with a user interface implemented in QML. It includes example assets, a simple UI, and a minimal audio controller.

## Features

Play local audio files
Basic playback controls: Play / Pause / Stop
QML-based user interface
Example audio assets
Built with CMake and Qt 6

## Requirements

Qt 6 (latest recommended)
C++17-compatible compiler
CMake 3.16+
Linux, Windows, or macOS (any platform supported by Qt)
